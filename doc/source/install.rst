.. _install:

Install
*******

Via Conda (installs depedencies for you)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: bash

    conda install -c conda-forge aneris

Via Pip
~~~~~~~

.. code-block:: bash

    pip install aneris-iamc

From Source (`master branch)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: bash

    pip install git+https://github.com/iiasa/aneris.git
